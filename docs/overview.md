# 项目总览

本项目是一个在线微信公众号文章编辑器。

用户在线编写Markdown格式的内容，系统会实时渲染为优美，简洁，统一风格的微信公众号文章HTML。

让用户可以专注内容创建，无需关注排版。

## 核心能力

### Markdown编辑器

在线Markdown编辑器，左侧 Markdown 编辑器，右侧实时预览。
使用remark/rehype将Markdown AST 转换为 HTML AST。
Markdown支持一键格式化，一键复制。
自动保存草稿。

### 样式主题

内置 3 到 5 套主题，用户可以根据需要选择。
切换样式主题后，会立即生效。
主题样式可以实时微调，字体，字号，颜色，间距等。
要支持代码块的样式。

### 微信公众号适配层

微信公众号文章必须是 inline style HTML，不能够使用 css class。
微信排版兼容性需要考虑。
生成的文章支持一键复制。
